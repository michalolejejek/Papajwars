Aby zagrać w grę musisz mieć zainstalowanego pythona w wersji 3

ponadto wymagany jest pakiet "pip3"

wymagane biblioteki do odpalenia gry

-pyglet
- playsound

instalowanie bibliotek przegia w taki sposób

piszemy w konsoli cmd tak:
1)pyglet

pip3 install pyglet //(proste co nie?)

2)playsound

pip3 install playsound

Czasem instalacja przebiega tak:

pip3 install nazwa_pakietu -Usr
